# Contributing to 段子工作台

感谢你考虑为段子工作台做出贡献！

## 开发环境

1. Fork 本仓库并 clone 到本地
2. 安装依赖：`npm install`
3. 启动开发模式：`npm start`

## 提交代码

1. 创建功能分支：`git checkout -b feature/my-feature`
2. 确保代码无 lint 错误
3. 提交时使用清晰的 commit message（中文或英文均可）
4. 推送并创建 Pull Request

## 代码风格

- 使用 ES6+ 语法
- 中文注释，变量名用英文
- CSS 使用项目中已定义的 CSS Variables
- 保持与现有代码风格一致

## Bug 报告

请使用 GitHub Issues 提交 Bug 报告，包含以下信息：

- macOS 版本
- Node.js / Electron 版本
- 复现步骤
- 期望行为与实际行为
- 截图（如有）

## 功能建议

欢迎在 Issues 中提出功能建议，请说明使用场景和预期效果。

## 许可证

贡献的代码将遵循项目的 [MIT License](LICENSE)。
